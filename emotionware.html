Emotionware

Smart wearables for Autism (TAMUHACK '26)

GitHub Live Project LinkedIn

Overview

Our system tracks emotions in real time using a webcam using DeepFace and Presage SDK (which will be Meta RayBans Smart Glasses in the Future), detects and assigns IDs to multiple faces, predicts their emotional state, and logs everything to a CSV. At the end of a session, the system sends the data to Google Gemini via OpenRouter to generate a clear, human-readable summary onto a secure dashboard. We also integrated wearable feedback. Using a glove, bracelet, ring, or even a smartwatch, the system can send vibrations corresponding to detected emotions. The feedback is customizable, so it could work in multiple form factors depending on the user’s needs. Finally, the web dashboard reads the CSV and displays a live session overview: dominant emotion, stability, and an AI-generated summary. You can even track trends over time, making it a powerful tool for both personal and educational use.

Technical Details

Python, Computer Vision, DeepFace, Presage SDK, Arduino/ESP32, Google Gemini API, OpenRouter, Flask, HTML/CSS/JS, Hardware Soldering, Team Collaboration

Challenges & Learnings
